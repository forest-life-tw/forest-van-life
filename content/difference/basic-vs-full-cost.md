## 車宿三寶：床面、電力、冷氣

車宿改裝的核心需求可以濃縮成三項：

- **床面**：睡得平、睡得穩，是一切的基礎
- **電力**：燈光、手機充電、電器用電，決定你在車裡能做什麼
- **冷氣**：台灣除了高海拔山區，其他地區夏天要住得舒適，冷氣幾乎是必要條件

DIY 入門只需要床面就能過夜。但真正讓車宿變成「舒適可持續的生活方式」，這三項缺一不可。

---

## 兩個層級的比較

### 基本車宿（DIY，目標：能睡覺）

**費用：0–3 萬**

| 項目 | 說明 |
|---|---|
| 隔熱材 | 車廂四壁與天花板 |
| 床板 | 平台式床板，可收折 |
| 捲簾 / 遮光布 | 隱私與遮陽 |

這個配置能完成最基本的過夜需求。電力靠行動電源、手機充電靠車用點菸器，夏天需停在陰涼處或外掛行動冷氣。適合想先試看看、或以 DIY 方式一步步升級的人。

---

### 完整改裝（目標：車廂即小宅）

**費用：30 萬起（依裝備選項，百萬以上亦有）**

| 項目 | 說明 |
|---|---|
| 全車隔熱 + 隔音 | 含地板、壁面、天花板 |
| 全木作床架 + 收納 | 客製化尺寸，附抽屜與上方吊櫃 |
| 後廚台（水槽 + 爐具空間） | 固定式，接水箱與排水 |
| 副電池 300AH 鋰電起 | DC-DC 充電 + 岸電雙充 |
| 太陽能板 500W | 日間補電，減少外部電源依賴 |
| 駐車分離式冷氣 | 獨立主機，不發動引擎可運行 |
| 12V 壓縮機冰箱（40L） | 長途儲食 |
| 天窗換氣扇 | 通風與空氣對流 |

三寶全數到位，再加上廚房、收納、冰箱，接近「移動小宅」等級，適合長途旅行或多日連住。

## 費用差距的關鍵在哪？

| 差距來源 | 費用影響 |
|---|---|
| 木作複雜度 | 全木作 vs 床板，差距達 8–15 萬 |
| 電力系統 | 行動電源 vs 固定副電池＋太陽能，差距 8–15 萬 |
| 冷氣方式 | 行動冷氣 2 萬 vs 分離式冷氣 5 萬＋搭配電池 |
| 廚房設備 | 攜帶式爐具 vs 固定廚台，差距 3–6 萬 |

**電力系統與冷氣是費用差距最大的環節**，佔了完整配置與基本配置差距的 50% 以上。

## 決策建議

**選基本 DIY，若你：**
- 想先體驗車宿，確認是否適合自己
- 預算有限，計劃分階段升級
- 主要做短天數假日車宿

**選完整配置，若你：**
- 計劃長途旅行或連住多天
- 重視舒適性，台灣夏天沒有冷氣住不住
- 一次到位，不想日後再改

## 常見問題

**Q：可以先做基本配置，之後再擴充嗎？**
A：可以，但需要在第一次施工時預留「擴充接口」（電路預留線、木作預留空間），否則後續擴充需大幅拆裝，反而增加費用。建議在初次設計時就跟廠商討論未來計劃。

**Q：分兩次施工比一次到位貴多少？**
A：通常貴 10–20%。施工費（人工、場地）重複支出是主要原因。若預算允許，一次規劃完整再分期施工更划算。
