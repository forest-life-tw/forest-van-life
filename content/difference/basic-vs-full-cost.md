## 兩個極端的比較

### 基本車宿配置（目標：能住、能睡）

**費用：5–15 萬**

| 項目 | 說明 |
|---|---|
| 隔熱材 | 車廂四壁與天花板 |
| 床板 | 平台式床板，可收折 |
| 小型行動電源 | 500–1000 Wh 行動電站 |
| 基本照明 | LED 燈條 |
| 捲簾 / 遮光布 | 隱私與遮陽 |

這個配置能完成基本車宿需求：過夜睡眠、手機充電、簡單照明。夏天需外掛行動冷氣或停在陰涼處，廚房功能靠攜帶式爐具解決。

---

### 完整改裝配置（目標：車廂即小宅）

**費用：40–80 萬**

| 項目 | 說明 |
|---|---|
| 全車隔熱 + 隔音 | 含地板、壁面、天花板 |
| 全木作床架 + 收納 | 客製化尺寸，附抽屜與上方吊櫃 |
| 後廚台（水槽 + 爐具空間） | 固定式，接水箱與排水 |
| 副電池 200 AH 鋰電 | DC-DC 充電 + 岸電雙充 |
| 駐車分離式冷氣 | 獨立主機，不發動引擎可運行 |
| 太陽能板（200W） | 日間補電 |
| 12V 壓縮機冰箱（40L） | 長途儲食 |
| 天窗換氣扇 | 通風與空氣對流 |

這個配置接近「移動小宅」等級，可以長住、自炊、夏天維持室溫，適合長途旅行或多日連住。

## 費用差距的關鍵在哪？

| 差距來源 | 費用影響 |
|---|---|
| 木作複雜度 | 全木作 vs 床板，差距達 8–15 萬 |
| 電池容量 | 行動電站 1 萬 vs 固定副電池 5 萬 |
| 冷氣方式 | 行動冷氣 2 萬 vs 分離式冷氣 5 萬+電池 |
| 廚房設備 | 攜帶式爐具 vs 固定廚台，差距 3–6 萬 |

**電力系統與冷氣是費用差距最大的環節**，佔了完整配置與基本配置差距的 50% 以上。

## 決策建議

**選基本配置，若你：**
- 主要做短天數假日車宿
- 喜歡極簡、保持靈活性
- 預算有限，日後可分階段升級

**選完整配置，若你：**
- 計劃長途旅行或連住多天
- 重視舒適性，不想將就
- 一次到位，不想日後再改

## 常見問題

**Q：可以先做基本配置，之後再擴充嗎？**
A：可以，但需要在第一次施工時預留「擴充接口」（電路預留線、木作預留空間），否則後續擴充需大幅拆裝，反而增加費用。建議在初次設計時就跟廠商討論未來計劃。

**Q：分兩次施工比一次到位貴多少？**
A：通常貴 10–20%。施工費（人工、場地）重複支出是主要原因。若預算允許，一次規劃完整再分期施工更划算。
