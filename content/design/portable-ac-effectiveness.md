## 行動冷氣的現實期待

先說結論：**行動冷氣在小型廂車 + 做好隔熱的前提下，夏天整晚使用是可行的**，但在大廂車或未做隔熱的情況下，效果有限。

## 影響效果的關鍵因素

### 1. 車廂大小
行動冷氣的冷量標示（BTU 或 Watt）是在「密閉空間」測量的。廂型車後廂越小，冷卻效果越明顯：

| 車廂空間 | 行動冷氣效果 |
|---|---|
| Caddy / Combo（小廂車） | 明顯，可有效降溫 |
| J-Space / Staria（中型廂車） | 尚可，需要足夠時間降溫 |
| Hiace（大廂車） | 有限，最熱的夏天可能僅能勉強維持 |

### 2. 隔熱情況
未做隔熱的鐵皮車廂散熱快，冷氣降下去的熱量很快從車壁補回來，冷氣機「白做工」。

做好隔熱後，同樣的冷氣機效果可以提升一倍以上。

### 3. 排熱效率
行動冷氣的熱端（排熱）管必須導出車外，若管子彎折過多、密封不良，熱空氣回流車廂內，製冷效果大打折扣。

### 4. 功率規格
台灣市場常見行動冷氣功率：

| 類型 | 功率 | 適合車廂 |
|---|---|---|
| 低功耗露營型 | 100–200W | 小至中型廂車 |
| 一般攜帶式 | 700–900W | 效果好但耗電大 |

> 車宿專用的低功耗行動冷氣（如 EcoFlow Wave 2、Zero Breeze Mark 2）功率約 1,000–1,500 BTU，耗電 150–300W，是目前平衡效果與電耗的主流選擇。

## 整晚耗電試算

以低功耗行動冷氣（150W）× 8 小時：

| 電量需求 | 適用電池 |
|---|---|
| 1,200 Wh（冷氣） | |
| + 480 Wh（冰箱） | |
| + 265 Wh（其他） | |
| **合計約 1,945 Wh** | **200 AH 鋰電（2,400 Wh）勉強夠** |

若加上太陽能補電，整天旅行結束後通常電量充足。

## 實際使用者常見回饋

- 「春秋兩季完全夠用，夏天 35°C 以上比較辛苦」
- 「車廂做了隔熱後，同一台行動冷氣效果明顯提升」
- 「噪音比分離式大，用耳塞睡覺」
- 「排熱管固定在窗縫，睡覺時車窗不能全關」

## 常見問題

**Q：行動冷氣和分離式冷氣的溫差大概多少？**
A：分離式冷氣可以輕鬆將 Hiace 車廂降至 22–24°C；行動冷氣在相同車廂可能只能降至 27–29°C（盛夏高溫情況下）。對於追求舒適睡眠的人，差異會感覺明顯。

**Q：行動冷氣的噪音影響睡眠嗎？**
A：功率較小的露營型行動冷氣（如 EcoFlow Wave 2）噪音約 44–52 dB，相當於一般對話音量。多數人適應一兩晚後不覺得困擾，但對噪音敏感的人可能需要搭配白噪音或耳塞。

**Q：行動冷氣用在 Hiace，效果如何？**
A：Hiace 車廂大，低功耗行動冷氣在夏天只能局部降溫（床鋪周圍），無法讓整個車廂感覺涼爽。若 Hiace 是主要車型，建議評估分離式冷氣或高功率行動冷氣。
