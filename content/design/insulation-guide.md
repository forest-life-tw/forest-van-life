## 隔熱與隔音是不同的工程

很多人混淆這兩個概念：

| 功能 | 目標 | 主要材料 |
|---|---|---|
| **隔熱** | 阻止熱量傳入車廂 | 反射材、閉孔發泡材 |
| **隔音** | 減少行駛噪音、路面噪音傳入 | 制震材、吸音棉、隔音氈 |

兩種需求需要不同材料，通常在車廂施工時一起進行，但不能用同一種材料同時解決兩個問題。

## 隔熱材料比較

### 鋁箔反射材
利用鋁箔反射輻射熱，是最便宜的入門選項。

| 項目 | 說明 |
|---|---|
| 原理 | 反射太陽輻射熱，非阻斷傳導熱 |
| 效果 | 中等，適合預算有限的入門方案 |
| 缺點 | 僅靠反射，對傳導熱效果有限；長期使用鋁箔可能受濕氣侵蝕 |
| 施工難度 | 低，剪貼即可 |
| 費用（全車） | 0.3–0.8 萬 |

### PE 發泡材（聚乙烯發泡）
最常見的廂車隔熱材，閉孔結構不吸水：

| 項目 | 說明 |
|---|---|
| 厚度建議 | 20–30mm |
| 效果 | 良好，兼顧隔熱與隔音（部分效果）|
| 優點 | 輕量、防水、施工方便、台灣容易取得 |
| 缺點 | 不如橡塑海綿完整，但 CP 值高 |
| 費用（全車） | 0.8–1.5 萬 |

### 橡塑海綿（類 Armaflex）
冷凍管線常用的保溫材料，是廂車隔熱的頂級選擇：

| 項目 | 說明 |
|---|---|
| 厚度建議 | 19–25mm |
| 效果 | 最佳，閉孔防水，導熱係數最低 |
| 優點 | 抑制結露、不吸水、防黴效果好、壽命長 |
| 缺點 | 費用較高，施工需仔細（接縫要貼緊）|
| 費用（全車） | 1.5–3 萬 |

## 隔音材料比較

### 制震貼片（STP / Dynamat 類）

貼在鐵板上，減少鐵皮共振噪音：

| 項目 | 說明 |
|---|---|
| 適用位置 | 車廂壁面鐵板、地板下金屬板 |
| 效果 | 消除「空鼓音」，行駛噪音明顯降低 |
| 施工 | 貼在裸鋁鐵皮上，需烙鐵壓實 |
| 費用（局部） | 0.5–1.5 萬 |

### 吸音棉（玻璃棉 / 隔音棉）

鋪在制震層上方，吸收殘餘聲音：

| 項目 | 說明 |
|---|---|
| 注意事項 | 選閉孔或防水材質，開孔型會積濕 |
| 適用位置 | 壁面（隔熱材外層）|
| 效果 | 搭配制震材後，整體隔音效果最佳 |

## 施工順序建議

廂車壁面的標準施工層次（由外到內）：

1. **車廂原始鐵板**
2. **制震貼片**（貼在鐵板上消除共振）
3. **隔熱材**（橡塑海綿或 PE 發泡，30mm）
4. **木作內裝**或飾板

地板的施工層次：

1. **金屬地板**
2. **制震材**
3. **木夾板**（12–15mm，防水）
4. **地板材**（防水地板貼）

## 常見問題

**Q：只做隔熱不做制震，行嗎？**
A：可以，但噪音會相對明顯。若你對行駛噪音不敏感，僅做隔熱也能解決夏天悶熱問題。

**Q：隔熱材厚度越厚越好嗎？**
A：有效益遞減：從 10mm 升到 25mm 效果提升明顯，但從 25mm 升到 50mm 提升幅度相對有限，卻佔用更多車廂空間。建議 20–30mm 是平衡點。

**Q：台灣氣候潮濕，隔熱材會不會長黴菌？**
A：開孔型材料（一般海綿、玻璃棉）容易積濕發霉；閉孔型材料（PE 發泡、橡塑海綿）不吸水，長期不發霉。台灣氣候強烈建議選閉孔材料。
