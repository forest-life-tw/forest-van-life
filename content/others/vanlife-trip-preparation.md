## 出發前確認清單

### 車輛狀況確認

- [ ] 引擎機油、冷卻水、煞車油確認在標準範圍
- [ ] 輪胎胎壓確認（含備胎）
- [ ] 雨刷、燈光功能正常
- [ ] 副電池電量充滿（或出發前行駛補電確認狀態）
- [ ] 冷媒檢查（若有駐車冷氣）

### 電力系統

- [ ] 副電池電量滿電出發
- [ ] 逆變器、充電器連接確認
- [ ] 岸電延長線帶了嗎（有停露營區時使用）
- [ ] 行動電源 / 行動電站滿電

### 水路系統（若有安裝）

- [ ] 水箱裝滿
- [ ] 水泵運作確認
- [ ] 排水桶清空
- [ ] 備用飲水（2L × 天數 × 人數，以防水箱不夠）

## 生活備品清單

### 飲食

- [ ] 便攜式爐具 + 瓦斯罐（確認容量，1 瓦斯罐約煮 3–4 餐）
- [ ] 鍋具（1 鍋多用最省空間）
- [ ] 餐具、筷子、開瓶器、小刀
- [ ] 保鮮盒（備料裝剩食）
- [ ] 調味料（小瓶分裝，鹽、醬油、油）
- [ ] 3 天份備糧（麵條、罐頭、乾糧，應急用）
- [ ] 飲用水

### 睡眠

- [ ] 寢具確認（床單、枕頭、棉被或睡袋）
- [ ] 遮光布 / 車窗遮光板（白天停車睡午覺用）
- [ ] 睡前用品（耳塞、眼罩）

### 盥洗

- [ ] 洗漱用品（牙刷、洗面乳、毛巾）
- [ ] 濕紙巾（無法淋浴時的應急清潔）
- [ ] 乾洗手
- [ ] 手提式折疊洗澡袋（簡易淋浴）
- [ ] 廁所：確認路線上的公廁分布；備用免洗內褲

### 衣物

- [ ] 按天數 + 1 天備用量帶衣物
- [ ] 防雨外套（台灣天氣多變）
- [ ] 夾腳拖 / 洞洞鞋（車廂內穿著）

## 應急備品

| 類別 | 建議備品 |
|---|---|
| 急救 | 基本急救包（OK 繃、消毒棉、繃帶、止痛藥）|
| 工具 | 隨身多功能工具鉗、手電筒、打火機 |
| 車用 | 警示三角架、反光背心、充氣泵（輪胎應急）|
| 行動電源 | 萬用充電線（iPhone + Type-C）|
| 地圖 | 離線地圖下載（Google Maps 離線；Maps.me）|
| 緊急聯絡 | 家人知道路線；手機電量低時的應對方案 |

## 出發前 24 小時確認事項

1. **天氣查詢**：確認目的地 3 天天氣，有颱風警報取消或改路線
2. **停車地點**：預先規劃第一晚停哪，不要臨時找
3. **通訊確認**：偏遠地區可能收訊差，告知家人大致行程
4. **車輛總重確認**：裝滿行李後不超過核定載重
5. **後廂整理**：物品固定，避免行駛中滑動或碰撞

## 常見問題

**Q：需要帶多少天的水？**
A：基本原則是每人每天 2 公升（飲用）+ 1 公升（烹飪 / 洗漱），3 人 3 天需備約 27 公升。若有水箱（20L），加上礦泉水備用，通常夠用 2–3 天。

**Q：第一次長途車宿，路線怎麼規劃？**
A：建議從 2–3 天的短程開始，選擇台灣熟悉地區（西海岸、花東）試跑，熟悉電力系統、水路使用、停車節奏後，再規劃環島或長程旅行。
